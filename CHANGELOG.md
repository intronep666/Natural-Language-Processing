## [1.0.0] - 2025-11-29

### Added

#### Documentation
- ✨ Comprehensive README with detailed NLP concepts and techniques
- ✨ GETTING_STARTED.md guide for setup and running notebooks
- ✨ CONTRIBUTING.md guidelines for contributors
- ✨ CHANGELOG.md to track version history
- ✨ .gitignore file for version control best practices
- ✨ requirements.txt with all dependencies

#### Core Practicals
- ✨ **01_Comprehensive_NLP_Pipeline_Linguistic_Analysis.ipynb**
  - Text preprocessing using spaCy and NLTK
  - Tokenization, POS tagging, lemmatization
  - Dependency parsing and Named Entity Recognition
  - Complete linguistic analysis pipeline

- ✨ **02_N_Gram_Analysis_Tokenization_Probability.ipynb**
  - N-gram generation (unigrams, bigrams, trigrams)
  - Frequency distribution analysis
  - Probability calculations
  - Language model fundamentals

- ✨ **03_Feature_Extraction_TF_TF-IDF.ipynb**
  - Term Frequency (TF) vectorization
  - TF-IDF implementation and comparison
  - Feature importance ranking
  - Sparse matrix operations

- ✨ **04_Word_Embeddings_Word2Vec_GloVe_FastText_BERT.ipynb**
  - Word2Vec (CBOW and Skip-gram)
  - GloVe embeddings
  - FastText for out-of-vocabulary words
  - BERT contextual embeddings
  - Semantic similarity exploration

- ✨ **05_Text_Classification_Naive_Bayes_SVM.ipynb**
  - Multinomial Naïve Bayes classifier
  - Support Vector Machine (SVM)
  - 20 Newsgroups dataset classification
  - Model evaluation and comparison

- ✨ **06_K-Means_Text_Clustering.ipynb**
  - Unsupervised K-Means clustering
  - TF-IDF vectorization for clustering
  - Cluster interpretation
  - Document similarity analysis

- ✨ **07_POS_Tagging_Part_of_Speech.ipynb**
  - Part-of-Speech tagging with NLTK
  - Grammatical role identification
  - Syntactic analysis
  - POS tag interpretation

- ✨ **08_Text_Processing_LSTM_Sentiment_Classification.ipynb**
  - Text preprocessing for neural networks
  - LSTM architecture implementation
  - Sentiment classification model
  - Sequence padding and embedding

- ✨ **09_Advanced_LSTM_Sentiment_Classification.ipynb**
  - Advanced LSTM with dropout regularization
  - Overfitting prevention techniques
  - Hyperparameter tuning
  - Model performance monitoring

- ✨ **10_Spam_Detection_Naive_Bayes_Application.ipynb**
  - Real-world NLP application
  - Spam vs. Ham classification
  - Bag-of-Words feature extraction
  - Practical preprocessing techniques
